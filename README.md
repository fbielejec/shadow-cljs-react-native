# [react-native-demo]( )
 
## Development

```
# first run : install the dependencies
$ yarn deps

# start watcher
$ yarn app:watch

# wait for first compile to finish or expo gets confused
$ yarn app:android
# or
$ yarn app:ios

# open the project in expo app (simulator or real device)

```
